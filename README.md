# Bevear Releases

This repository hosts official release builds (APK/IPA) for **Bevear** — a smart personal budget companion app for tracking spending, managing goals, and staying on top of your finances.

## Download

Check the [Releases](../../releases) tab for the latest APK (Android) and IPA (iOS) builds.

## Installation

**Android:** Download the `.apk` file and enable "Install from unknown sources" in your device settings.

**iOS:** Requires sideloading via Sideloadly or AltStore since Bevear is not yet on the App Store.

## Note

This repository does not contain the app's source code — only release builds are published here.
